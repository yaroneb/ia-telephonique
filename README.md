# IA Téléphonique - Système de Réponse Automatique

Système d'IA conversationnelle pour répondre aux appels téléphoniques via WebSocket.

## Stack Technique
- **STT** : OpenAI Whisper
- **IA** : Groq (Llama 3.3 70B)
- **TTS** : ElevenLabs
- **Backend** : FastAPI + WebSocket
- **Hébergement** : Render (gratuit)
